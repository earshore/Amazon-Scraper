## 3. Happy Path：US / UK / DE

对每个站点至少测 **3** 个商品详情页。分析成功后核对预览与导出 JSON。

**通用通过标准（字段级）**

| 字段 / 行为 | 通过标准 |
|-------------|----------|
| `title` / `productTitle` | 非空，与页面标题实质一致 |
| `asin` | 与 URL 中 ASIN 一致，非 `UNKNOWN` |
| `price` | 有合理价格字符串（页面无标价时记录备注，并核对 warnings/notes） |
| `brand` | 有品牌或合理回退；异常时有警告说明 |
| `main_image` | 主图 URL 非空且可访问（或懒加载失败时有警告） |
| `feature_bullets` | **≥ 1** 条（页面确实有卖点时） |
| `customer_reviews` | **0 条允许**（仅可见 DOM；需在 notes/备注说明） |
| `metadata.schema_version` | 固定 **`1.3.0`** |
| `metadata.reviews_scope` | 固定 **`visible_dom_only`** |
| `notes` / `warnings` | 行为符合第 4 节语义；无误导性「全量评论」表述 |

**导出 JSON 建议快速核对路径**

1. 点击「分析此页面」→ 等待完成  
2. 导出 JSON  
3. 确认：`metadata.schema_version === "1.3.0"`  
4. 确认：`metadata.reviews_scope === "visible_dom_only"`  
5. 确认：`products[0]` 含 title/asin/price/brand/main_image/feature_bullets/customer_reviews/scrape_status/coverage/warnings（及 notes，若本版本导出）

## 4. 状态语义（`scrape_status` / notes / warnings）

在可控场景下验证状态判定（优先用真实页面或临时 DOM 改动；改动后请刷新还原）。

| # | 场景 | 构造方式（建议） | 期望 | 结果 | 备注 |
|---|------|------------------|------|------|------|
| 4.1 | 缺少标题 → **failed** | 商品页上通过 DevTools 移除/清空标题节点后分析；或使用无法识别标题的异常页 | `scrape_status === "failed"`；有明确失败提示；正常失败路径下导出 JSON应不可用或不作为成功结果 | ☐ Pass / ☐ Fail | |
| 4.2 | 有标题、**仅**缺评论 → 可为 **success + notes** | 选一有标题/价格/品牌/主图/卖点齐全、但当前 DOM 无可见评论（或评论区未加载）的商品；确认无其他 warnings | **不得**仅因「无可见评论」判为 `partial`；可为 `success`，并以 **notes**（或等价非阻断说明）记录「仅可见 DOM / 当前无评论」；`customer_reviews` 可为 `[]` | ☐ Pass / ☐ Fail | |
| 4.3 | 页面有卖点但 bullets 全漏抓 → **warning / partial** | 肉眼确认 About this item / 卖点列表存在，但导出 `feature_bullets` 为空（选择器失效或被隐藏） | 出现描述点相关 **warning**；`scrape_status` 为 **`partial`**（有标题时）；预览 coverage 显示描述点异常 | ☐ Pass / ☐ Fail | |
| 4.4 | 描述点偏少（可选） | 仅抓到 1–2 条卖点 | 可有「描述点偏少」类警告，状态为 `partial`（若本版本仍将此类计入 warnings） | ☐ Pass / ☐ Fail / ☐ Skip | |
| 4.5 | 缺价格/品牌/主图（可选） | 无标价或区域缺失的商品 | 记入 **notes**（说明），**不**因此单独强制 `partial`；有标题且无卖点类 warnings 时可为 `success`；仍可导出 JSON | ☐ Pass / ☐ Fail / ☐ Skip | |

**判定摘要（发版期望，schema 1.3.0）**

| 条件 | 期望状态 |
|------|----------|
| 无标题 | `failed`（`errors`） |
| 有标题 + 无 `warnings`（评论为 0 / 无价格等仅 `notes`） | `success` |
| 有标题 + 存在质量类 `warnings`（如无卖点、卖点偏少、ASIN 未知） | `partial` |
| 仅无可见评论 / 仅无价格品牌主图 | **不**单独强制 `partial`；优先 `success` + notes |

| 测试人 | 日期 | 总评 |
|--------|------|------|
| | | ☐ Pass / ☐ Fail |
